##### Yep. Files

### Usage:
# Install git
- Arch:
```
sudo pacman -S git
```
# Install other stuff (optional):
- Arch:
```
sudo pacman -S cava neofetch starship neovim spotify spicetify
```

# Clone and run setup.sh:
```
git clone https://github.com/DesantEu/dotfiles
cd dotfiles
chmod +x setup.sh
./setup.sh
```
